Library Managment System

LMS - is a system to manage library, created as part of an internship. In system you can search authors and information about them. You can also display all books stored in database. User with admin role can delete selected author or book. Without this role can only add and update previously mentioned books and authors. In LMS available also option to sort searched result.

Technologies used:

  • Back-end
    • JAVA
    • Spring Boot

  • Front-end
    • HTML
    • CSS
    • Angular
  • Database
    • PostgreSQL
    • PgAdmin

  • Other
    • Postman
